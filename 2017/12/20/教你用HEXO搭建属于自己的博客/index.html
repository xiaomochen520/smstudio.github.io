教你用HEXO搭建属于自己的博客

2017-12-20 阅读量

我们平时遇到问题时，就回去百度搜索相关问题，就会搜到很多知名网站的帖子，有很多大牛在里面分享经验，只是。久了，不知道有没有小伙伴想要自己弄一个博客，但是又不知道如何开始。那么本篇文章就是为你准备的。

HEXO 是什么？

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

为什么要使用HEXO？

​ 是的，为什么要使用HEXO，WordPress、Discuz等知名平台不可以搭建个人博客么？答案当然是可以，而且也不是很复杂。但是，你把本文看完了，我相信这个问题你也就有答案了。

如何使用HEXO?

安装

Hexo 依赖NodeJS，Git

安装NodeJS、Git

如果你没有接触过Node’JS，也请照着步骤做

安装NodeJS

  1. 打开 NodeJS官网下载页面 ，选择有LTS字样的版本下载，也可以点击 这里 这个连接是官网的下载连接，也可以去我的文件站 下载。

  2. 打开刚刚下载的文件，双击它，然后一路下一步到底。

  3. 打开cmd窗口，输入 node -v 返回给你node的版本号就是已经安装成功了。
